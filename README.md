# Crypto Detector

Crypto bot that detect new coin listing on exchanges and send buy/sell order taking advantage of the "listing pump".

Supported exchanges:

- [Binance](https://www.binance.com/)

Supported deployment modes:

- [AWS](https://aws.amazon.com/)

## Getting started

### Requirements

- NodeJS 14.x
- Yarn 1.x
- AWS SAM 1.x
